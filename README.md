# Sayyad Yasmin — Portfolio

A modern and responsive personal portfolio built using **React**, **Vite**, and **JavaScript** to showcase my skills, projects, certifications, achievements, and resume.

## Tech Stack
- React
- Vite
- JavaScript
- CSS
- Framer Motion
- React Icons
- EmailJS

## Features
- Responsive single-page design
- Smooth animations
- Projects and skills showcase
- Resume section
- Contact form with EmailJS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

This portfolio is deployed on **Vercel** and is optimized for desktop and mobile devices.
